Problem Statement : 
EV market pricing varies widely. Businesses need insights into factors influencing pricing and demand.

Dataset : 
10K+ records collected via web scraping
Features: brand, range, battery, price
Tools Used
Python (Requests, BeautifulSoup, Pandas)
Data Cleaning & EDA


Approach :
Built data pipeline to scrape EV data
Cleaned and structured raw data
Performed correlation and multivariate analysis
Identified key pricing drivers


Key Insights :
Battery capacity and driving range strongly influence pricing
Premium brands dominate high-price segments
Mid-range EVs show highest demand concentration
Business Impact
Helps pricing strategy decisions
Supports market positioning
Identifies competitive gaps
